#Diagnosis of Autism Spectrum Disorder using Machine Learning Techniques

## Project Description

This project focuses on the diagnosis of Autism Spectrum Disorder (ASD) using machine learning techniques. It analyzes child behavioral data and applies multiple machine learning algorithms to predict whether a child shows signs of ASD. The system provides prediction results based on the given input data.

## Algorithms Used

* Decision Tree
* K-Nearest Neighbors (KNN)
* Logistic Regression
* Naive Bayes
* Support Vector Machine (SVM)

## Dataset

The project uses a child behavioral dataset containing features related to social interaction, communication, and behavioral patterns to determine the likelihood of Autism Spectrum Disorder.

## Technologies Used

* Python
* Machine Learning
* Pandas
* NumPy
* Scikit-learn

## How to Run the Project

1. Install required libraries.
2. Load the dataset.
3. Train the machine learning models.
4. Provide input values.
5. View the prediction output.

## Output

The system predicts whether the child shows signs of Autism Spectrum Disorder based on the input data using trained machine learning models.
